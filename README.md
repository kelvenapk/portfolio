## Projetos em Destaque

### Windows Ultimate Optimizer
Script PowerShell completo para otimização extrema do Windows:
- Limpeza automática de arquivos temporários
- Desativação de telemetria e serviços desnecessários
- Otimização de rede para gaming
- Configurações de energia para performance máxima

### Gamer Performance Suite
Conjunto de otimizações específicas para gaming:
- Priorização de processos em tempo real
- Otimização de GPU e drivers
- Configurações de rede para baixa latência
- Modo game automático

### Privacy & Security Pack
Proteção avançada de privacidade:
- Bloqueio de telemetria do Windows
- Remoção de apps espiões
- Firewall rules otimizadas
- Proteção contra tracking
